# WanderLust ✈️

WanderLust is a full-stack web application inspired by Airbnb, designed to allow users to list, explore, and book unique accommodations around the world. This project was developed as part of the **Sigma 3.0 (Development)** course by Apna College.

## 🚀 Features

- **User Authentication & Authorization:** Secure signup, login, and logout functionalities with protected routes for managing listings.
- **Listing Management:** Full CRUD capabilities allowing users to create, view, update, and delete travel listings.
- **Review System:** Logged-in users can leave ratings and textual reviews for specific listings, as well as delete their own reviews.
- **Interactive Maps:** Visualized location tracking integrated directly into listing details.
- **Image Uploads:** Cloud-based image storage management for property pictures.
- **Responsive Design:** A fully adaptive UI styled with Bootstrap, optimized for mobile, tablet, and desktop viewports.

---

## 🛠️ Tech Stack

**Frontend:**
- HTML5 & CSS3
- JavaScript (ES6+)
- Bootstrap 5 (Styling & Layouts)
- EJS (Embedded JavaScript templates)

**Backend:**
- Node.js
- Express.js (Web Framework)

**Database & Cloud Services:**
- MongoDB & MongoDB Atlas (NoSQL Database)
- Mongo Session Store (Session Persistence)
- Cloudinary / Image Hosting (For listing images)
